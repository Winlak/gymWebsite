#gymWebsite

Реализована страница тарифов с:

- загрузкой тарифов из сервиса (`/api/tariffs` -> `https://t-core.fit-hub.pro/Test/GetTariffs`);
- дефолтным выбором тарифа по `is_best`;
- выделением выбранного тарифа;
- мигающей кнопкой `Купить` (без hover/transition-эффектов);
- проверкой чекбокса перед покупкой (ошибка подсвечивается красным);
- закрепленным хедером с таймером `02:00`;
- миганием и красной подсветкой таймера на последних 30 сек;
- исчезновением скидочных цен по завершении таймера с анимацией.

## Запуск

```bash
npm install
npm run dev
```

## Идея анимации после окончания таймера

Сейчас применена схема:

1. Блок со скидкой плавно исчезает (`discountOut`).
2. После исчезновения появляется блок с полной ценой (`priceIn`).

Альтернативный вариант: добавить тонкую красную плашку `Скидка завершена` поверх карточек на 1-1.5 сек и синхронно менять цену через crossfade.
